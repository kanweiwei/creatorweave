# 对话功能

与 AI 进行自然语言对话，完成各种开发任务。

## 对话管理

### 创建新对话

1. 点击"新建对话"按钮
2. 输入对话描述（可选）
3. 开始与 AI 交流

### 切换对话

1. 在左侧对话列表中点击任意对话
2. 当前对话会自动加载

### 对话历史

- 对话自动保存，无需手动保存
- 支持重命名对话方便识别
- 长对话会自动压缩上下文

## 发送消息

### 基本用法

在底部输入框中输入你的需求：

```
帮我创建一个用户登录表单
```

### 引用文件

使用 `@` 引用项目中的文件：

```
@src/utils/auth.ts
这个工具函数怎么使用？
```

### 代码片段

在消息中包含代码：

```
用这个接口写一个获取用户列表的函数：

```typescript
interface User {
  id: string;
  name: string;
}
```
```

## AI 响应

### 响应内容

AI 响应可能包含：

- **文字说明**: 解释正在做什么
- **代码块**: 生成的代码
- **文件变更**: 创建或修改的文件
- **执行结果**: 命令执行输出

### 确认执行

对于有风险的操作，AI 会请求确认：

- 删除文件
- 修改多个文件
- 执行破坏性命令

## 消息操作

### 编辑消息

- 点击已发送的消息
- 修改内容后重新发送
- AI 会基于修改后的内容响应

### 复制内容

- 点击代码块右上角的复制按钮
- 快速复制所需代码

### 重新生成

如果对响应不满意：

1. 点击消息底部的"重新生成"
2. AI 会重新思考并给出新答案

## 提升对话效果

### 清晰描述需求

❌ "帮我弄一下"

✅ "创建一个用户注册表单，包含邮箱、密码、确认密码三个字段"

### 提供上下文

❌ "这段代码报错"

✅ "这段代码在处理 JSON 解析时报错，错误信息是..."

### 分步骤进行

对于复杂任务，可以分步骤进行：

1. "先帮我设计数据库结构"
2. "基于这个结构创建数据模型"
3. "生成 CRUD API"
